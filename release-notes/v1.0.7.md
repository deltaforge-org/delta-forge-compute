## What's Changed

The DeltaForge compute node: the query-executing process. It connects to a DeltaForge control plane and runs queries on a single node. Scale out by running more nodes.

### GPU-accelerated graph analytics

- Graph algorithms execute on the GPU end to end, with host preprocessing moved onto the device. Community detection (Louvain and Leiden), connected components, strongly connected components, triangle counting, local clustering coefficient, betweenness centrality, eigenvector centrality, and FastRP embeddings all run on-device, returning columnar results.

### Stability

- Fixed a query hang caused by the schema observer's flush blocking the query worker.
- Fixed a control-plane connection hang from Postgres pool exhaustion under nested checkouts, along with related HTTP request-hang issues.
